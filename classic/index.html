The Classic BSP Interface¶

Since BSP originally was a C library, the whole interface was designed to be used from C. We made the interface easier to work with from C++, but we also provide the classic interface implementation.

Program Logic¶

Classic Modern
bsp_init() BSPLib::Classic::Init()
bsp_begin() BSPLib::Classic::Begin()
bsp_end() BSPLib::Classic::End()
